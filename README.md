# Run in Development
1. Clone the repository
2. Execute
```bash
yarn install
```
3. Should have Nest CLI installed
```
npm i -g @nestjs/cli
```
4. Build up the Database
```
docker-compose up -d
```
## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
## Stack used
* Mongo DB
* Nest
